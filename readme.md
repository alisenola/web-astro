# Web

Static web built with [Middleman](https://middlemanapp.com/). Hosted using [GitHub Pages](https://pages.github.com/).

## Building notes

1. Get Ruby
2. Get RubyGems
3. Get Bundler `gem install bundler`
4. Run `bundle install`
5. Run `bundle exec middleman build`
