# Web

Personal static web built with [Middleman](middlemanapp.com) and [Slim](slim-lang.com).

## Building notes

1. Get Ruby
2. Get RubyGems
3. Get Bundler `gem install bundler`
4. Run `bundle install`
5. Run `bundle exec middleman build`
